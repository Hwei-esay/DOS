GitHub Pages部署教程

创建GitHub账户

如果您还没有GitHub账户，请先访问 github.com 注册一个免费账户。

创建新的代码仓库

登录GitHub后，点击右上角的"+"图标，选择"New repository"。仓库名必须格式为：你的用户名.github.io

上传网站文件

将您的网站文件（HTML、CSS、JS等）上传到该仓库。您可以使用Git命令行或直接在GitHub网站上上传。

访问您的网站

上传完成后，访问 https://你的用户名.github.io 即可查看您的网站。最初可能需要几分钟才能生效。
